## 특징

- 캘린더 그리드·범례·연간 합계는 Kibo UI [Contribution Graph](https://www.kibo-ui.com/components/contribution-graph)를 기반으로 만들었어요.
- 기여 데이터는 Jonathan Gruber의 [GitHub Contributions API](https://github.com/grubersjoe/github-contributions-api)에서 불러와요.
- Next.js `unstable_cache`로 하루 동안 캐시하고, `Promise`를 클라이언트에 넘겨 React `use()`로 풀어요.

처음이라면 components.json에 `@smlee` 레지스트리를 한 번만 등록하세요.

```json
{
  "registries": {
    "@smlee": "https://smlee.info/r/{name}.json"
  }
}
```

그다음 컴포넌트를 추가해요.

```bash
npx shadcn@latest add @smlee/github-contributions
```
