## 특징

- Android에서는 [Vibration API](https://developer.mozilla.org/docs/Web/API/Vibration_API)를, iOS에서는 체크박스 트릭을 사용해요.
- Android 기기에서는 진동 시간과 패턴을 자유롭게 지정할 수 있어요.
- [Vercel](https://vercel.com) 모바일 앱의 햅틱 경험에서 영감을 받았어요.

```json
  }

```

그다음 컴포넌트를 추가해요.

```bash
npx shadcn@latest add @smlee/haptic
```

## 사용법

```tsx
import { haptic } from "@/lib/haptic"
```

```tsx
<Button onClick={() => haptic()}>Haptic</Button>
```

<Callout>
  햅틱 피드백을 아무 곳에나 쓰지 않는 게 좋아요. 언제 쓰는 게 적절한지는 Apple
  Human Interface Guidelines의 [Playing
  haptics](https://developer.apple.com/design/human-interface-guidelines/playing-haptics)
  문서를 참고하세요.
</Callout>
