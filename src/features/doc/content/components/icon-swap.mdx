## 사용법

```tsx
import { IconSwap, IconSwapItem } from "@/components/icon-swap"
```

```tsx
<IconSwap>
  <IconSwapItem key={isCopied ? "check" : "copy"}>
    {isCopied ? <CheckIcon /> : <CopyIcon />}
  </IconSwapItem>
</IconSwap>
```

아이콘을 교체하고 싶은 시점에 `IconSwapItem`의 `key`를 바꿔주세요.

## 구성

`IconSwap`은 다음 구조로 조합해요.

```text
IconSwap
└── IconSwapItem
    └── {icon}
```
