## 특징

- 세 가지 잘라내기 전략을 지원해요: 끝 글자 수 고정, 최소 글자 수 보장, 균등 분할.
- 실제로 렌더링된 텍스트 너비를 기준으로 픽셀 단위까지 정확하게 계산해요.
- 컨테이너 크기가 바뀌면 자동으로 다시 계산해요.

## 예시

### 끝 글자 수 고정

끝에서 지정한 글자 수를 항상 그대로 남겨요.

```tsx
<MiddleTruncation end={4}>
  FY26_Q1_Consolidated_Financial_Statements.pdf
</MiddleTruncation>
```

### 최소 글자 수 보장

텍스트를 균등하게 나누되, 끝에 최소한의 글자 수는 보장해요.

```tsx
<MiddleTruncation minEnd={12}>
  /Users/smlee/Code/smlee.info/node_modules/shadcn/package.json
</MiddleTruncation>
```

### 균등 분할

아무 조건도 주지 않으면 가운데를 기준으로 균등하게 나눠요.

```tsx
<MiddleTruncation>
  /Users/smlee/Code/smlee.info/src/components/ui/button.tsx
</MiddleTruncation>
```
