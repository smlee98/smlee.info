## 사용법

```tsx
import { useRef } from "react"

import {
  ChevronsUpDownIcon,
  ChevronsUpDownIconHandle,
} from "@/components/chevrons-up-down-icon"
```

```tsx
const chevronsUpDownIconRef = useRef<ChevronsUpDownIconHandle>(null)
```

```tsx
<ChevronsUpDownIcon ref={chevronsUpDownIconRef} />
```

ref를 통해 원하는 시점에 모프 애니메이션을 직접 실행할 수 있어요.

```tsx
chevronsUpDownIconRef.current?.startAnimation() // chevrons-down-up 방향으로 전환
chevronsUpDownIconRef.current?.stopAnimation() // chevrons-up-down 방향으로 복귀
```
